# My Cloud - Облачное хранилище

Fullstack веб-приложение для хранения и управления файлами в облаке.

## 🚀 Функциональность

### Для пользователей
- 📁 Загрузка и скачивание файлов
- 🔒 Безопасное хранение данных
- 👥 Публичные ссылки для обмена
- 📊 Статистика использования
- ✏️ Управление файлами (переименование, удаление, комментарии)

### Для администраторов
- 👥 Управление пользователями
- 🛠️ Назначение прав администратора
- 📈 Общая статистика системы
- 🔍 Просмотр хранилищ пользователей

## 🛠️ Технологии

### Backend
- Python 3.11
- Django 4.2
- Django REST Framework
- PostgreSQL
- Gunicorn
- Nginx

### Frontend
- React 18
- Redux Toolkit
- React Router
- Axios
